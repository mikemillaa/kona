Kona Node CLI Reference

This document lists all CLI flags for the kona-node node subcommand, grouped by category. All flags can be provided as command-line arguments or via environment variables.

Default Ports

ServiceDefault PortFlag/Env
RPC HTTP9545--rpc.port / KONA_NODE_RPC_PORT
RPC WebSocket9545(same as HTTP, enabled with --rpc.ws-enabled)
P2P TCP9222--p2p.listen.tcp / KONA_NODE_P2P_LISTEN_TCP_PORT
P2P UDP9223--p2p.listen.udp / KONA_NODE_P2P_LISTEN_UDP_PORT
Supervisor RPC9333--supervisor.port / KONA_NODE_SUPERVISOR_PORT
Conductor RPC8547--conductor.rpc / KONA_NODE_CONDUCTOR_RPC
